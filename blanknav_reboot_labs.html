Watson Labs

The Watson Project is a set of interactive labs implemented in Java, designed as a companion to introductory computer science courses. The labs are meant to reinforce fundamental computer science concepts through interactive examples and structured activities. Labs in Watson will cover the following subjects:
  • spreadsheet and database applications
  • data structures
  • computer graphics
  • JavaScript-based programming
  • object-oriented programming concepts
  • assembly language and machine architecture
  • digital logic
